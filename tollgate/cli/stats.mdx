# tollgate stats

Displays aggregate statistics from the Tollgate audit database.

## Usage

```bash
tollgate stats [options]
```

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--audit-path <path>` | Custom path for audit database | `~/.dotset/tollgate.db` |

## Examples

```bash
# Show statistics
tollgate stats

# Use custom audit database
tollgate stats --audit-path /var/log/tollgate.db
```

## Use Cases

### Security Auditing

Review overall policy effectiveness:

```bash
tollgate stats
```

### Capacity Planning

Understand usage patterns for scaling decisions.

### Policy Tuning

High denial rates might indicate overly restrictive policies. High prompt rates could suggest opportunities for automated allow rules.
