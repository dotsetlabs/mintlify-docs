# Tachyon Quickstart

**Zero-Trust Tunnels for Teams.** Share local development servers with built-in authentication.

## Installation

### CLI

```bash
npm install -g @dotsetlabs/cli
```

### SDK (for programmatic access)

```bash
npm install @dotsetlabs/tachyon
```

## Sign In

```bash
dotset login
```

Sign in with GitHub or Google SSO.

## Share a Local Port

```bash
dotset share 3000
```

You'll get a public URL like:

```
https://abc123.tunnel.dotsetlabs.dev
```

## Access Control

By default, tunnels require authentication. Share with your team:

```bash
# Public (no auth required)
dotset share 3000 --public

# Allow specific emails
dotset share 3000 --allow colleague@example.com

# Custom subdomain (Pro)
dotset share 3000 --subdomain my-app
```

## Projects

Organize tunnels in projects for the dashboard:

```bash
# Link to existing project
dotset share 3000 --project <project-id>
```

## Request Inspector

Inspect incoming requests in real-time:

```bash
dotset share 3000 --inspect
```

Opens the inspector at `http://localhost:4040`.
