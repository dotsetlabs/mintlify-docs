# Quickstart

This guide gets you from zero to protected in under a minute.

## 1. Install

```bash
npm install -g @dotsetlabs/shield
```

## 2. Zero-Migration Setup

Shield automatically detects `.env` files. If you already have one, you're ready to go:

```bash
# Run with automatic protection
dotset run -- npm start
```

## 3. Using Multiple Providers

Shield can aggregate secrets from multiple sources. For example, to use both `.env` files and AWS Secrets Manager:

```bash
dotset run --providers dotenv,aws-sm -- node server.js
```

## 4. Protection Modes

| Mode | Behavior | Use Case |
|:-----|:---------|:---------|
| `detect` | Warning in logs | Auditing |
| `redact` | Replaces secrets (default) | **Recommended** |
| `block` | Suppresses lines | High Privacy |

```bash
# Redact secrets in real-time
dotset run --mode redact -- npm test
```
