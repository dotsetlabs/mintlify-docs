# Quickstart

This guide gets you from zero to protected in under a minute.

---

## Mantle

Mantle protects your CI/CD pipelines from secret leakage.

### 1. Install

```bash
npm install -g @dotsetlabs/mantle
```

### 2. Zero-Migration Setup

Mantle automatically detects `.env` files. If you already have one, you're ready to go:

```bash
# Run with automatic protection
mantle run -- npm start
```

### 3. Protection Modes

| Mode | Behavior | Use Case |
|:-----|:---------|:---------|
| `detect` | Warning in logs | Auditing |
| `redact` | Replaces secrets (default) | **Recommended** |
| `block` | Suppresses lines | High Privacy |

```bash
# Redact secrets in real-time
mantle run --mode redact -- npm test
```

### 4. Example Output

```bash
$ mantle run --mode redact -- node app.js

mantle | mode: redact | secrets: 5 | providers: dotenv, environment

Starting server...
Connecting to database: [REDACTED]
Using API key: [REDACTED]
Server running on port 3000

mantle | exit: 0 | exposures: 2
```

## Tollgate

Tollgate adds policy-based access control to MCP servers used by AI agents.

### 1. Install

```bash
npm install -g @dotsetlabs/tollgate
```

### 2. Wrap Any MCP Server

Instantly protect any MCP server without configuration:

```bash
tollgate wrap npx @modelcontextprotocol/server-postgres
```

This prompts you for approval on every tool call.

### 3. Default Actions

| Action | Behavior | Use Case |
|:-------|:---------|:---------|
| `allow` | Proceed without approval | Trusted operations |
| `deny` | Block entirely | Dangerous operations |
| `prompt` | Ask user (default) | **Recommended** |

```bash
# Allow reads, prompt for everything else
tollgate wrap -d prompt npx @anthropic/mcp-server-filesystem ./src
```

### 4. Use with Claude Desktop

Update your Claude Desktop config:

```json
{
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": [
        "-y", "@dotsetlabs/tollgate", "wrap",
        "npx", "-y", "@modelcontextprotocol/server-postgres"
      ],
      "env": {
        "DATABASE_URL": "postgres://localhost/mydb"
      }
    }
  }
}
```

### 5. Create a Policy File (Optional)

For fine-grained control:

```bash
# Generate config
tollgate init

# Start with config
tollgate start -s postgres
```
