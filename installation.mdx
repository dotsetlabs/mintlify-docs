# Installation

## CLI (Mantle)

### npm (Recommended)

```bash
npm install -g @dotsetlabs/cli
```

### Yarn

```bash
yarn global add @dotsetlabs/cli
```

### pnpm

```bash
pnpm add -g @dotsetlabs/cli
```

### Verify Installation

```bash
dotset --version
```

## Requirements

- **CLI**: Node.js 20 or later

## Quick Start

```bash
cd your-project
dotset mantle run -- npm start
```

## Updating

```bash
npm update -g @dotsetlabs/cli
```

## Uninstalling

```bash
npm uninstall -g @dotsetlabs/cli
```
