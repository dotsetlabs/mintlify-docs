# Installation

## Hardpoint (Static Scanner)

Hardpoint is a Go binary that scans for Rules File Backdoor attacks and other AI-era security threats.

### Go Install (Recommended)

```bash
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest
```

### Download Binary

Download the latest release from [GitHub Releases](https://github.com/dotsetlabs/hardpoint/releases).

### Verify Installation

```bash
hardpoint --version
```

### Quick Start

```bash
hardpoint scan
```

---

## Overwatch (Runtime Security)

Overwatch is a Node.js package that provides runtime protection for MCP servers with tool shadowing detection.

### npm (Recommended)

```bash
npm install -g @dotsetlabs/overwatch
```

### Yarn

```bash
yarn global add @dotsetlabs/overwatch
```

### pnpm

```bash
pnpm add -g @dotsetlabs/overwatch
```

### Verify Installation

```bash
overwatch --version
```

### Quick Start

```bash
# Wrap an MCP server
overwatch wrap npx @modelcontextprotocol/server-postgres

# Or initialize configuration
overwatch init
```

## Requirements

| Tool | Requirements |
|:-----|:-------------|
| **Hardpoint** | Go 1.21+ (for building from source) |
| **Overwatch** | Node.js 20+ |

---

## Updating

```bash
# Hardpoint (reinstall latest)
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest

# Overwatch
npm update -g @dotsetlabs/overwatch
```

---

## Uninstalling

```bash
# Hardpoint (remove binary from GOPATH)
rm $(go env GOPATH)/bin/hardpoint

# Overwatch
npm uninstall -g @dotsetlabs/overwatch
```
