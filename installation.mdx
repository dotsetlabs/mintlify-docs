# Installation

Tollgate and Hardpoint can be installed independently based on your needs.

---

## Tollgate (MCP Access Control)

### npm (Recommended)

```bash
npm install -g @dotsetlabs/tollgate
```

### Yarn

```bash
yarn global add @dotsetlabs/tollgate
```

### pnpm

```bash
pnpm add -g @dotsetlabs/tollgate
```

### Verify Installation

```bash
tollgate --version
```

### Quick Start

```bash
tollgate wrap npx @modelcontextprotocol/server-postgres
```

---

## Hardpoint (Dev Environment Scanner)

### Homebrew (macOS / Linux)

```bash
brew install dotsetlabs/tap/hardpoint
```

### Go Install

```bash
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest
```

### Download Binary

Download the latest release from [GitHub Releases](https://github.com/dotsetlabs/hardpoint/releases).

### Verify Installation

```bash
hardpoint --version
```

### Quick Start

```bash
hardpoint scan
```

---

## Requirements

- **Tollgate**: Node.js 20 or later
- **Hardpoint**: Go 1.21 or later (for building from source)

## Updating

```bash
# Tollgate
npm update -g @dotsetlabs/tollgate

# Hardpoint (Homebrew)
brew upgrade dotsetlabs/tap/hardpoint
```

## Uninstalling

```bash
# Tollgate
npm uninstall -g @dotsetlabs/tollgate

# Hardpoint (Homebrew)
brew uninstall hardpoint
```
