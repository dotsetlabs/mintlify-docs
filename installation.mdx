# Installation

## Unified CLI (Recommended)

The easiest way to use all Dotset Labs tools is through the unified CLI:

```bash
npm install -g @dotsetlabs/dotset
```

Then install the Go binaries:

```bash
dotset doctor --install
```

This installs and manages Hardpoint, Tollgate, and Deadfall together. See the [Dotset CLI documentation](/dotset/cli/overview) for more details.

---

## Individual Tools

Dotset Labs tools can also be installed independently based on your needs.

---

## Hardpoint (Dev Environment Scanner)

### Go Install

```bash
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest
```

### Download Binary

Download the latest release from [GitHub Releases](https://github.com/dotsetlabs/hardpoint/releases).

### Verify Installation

```bash
hardpoint --version
```

### Quick Start

```bash
hardpoint scan
```

---

## Tollgate (MCP Access Control)

### npm (Recommended)

```bash
npm install -g @dotsetlabs/tollgate
```

### Yarn

```bash
yarn global add @dotsetlabs/tollgate
```

### pnpm

```bash
pnpm add -g @dotsetlabs/tollgate
```

### Verify Installation

```bash
tollgate --version
```

### Quick Start

```bash
tollgate wrap npx @modelcontextprotocol/server-postgres
```

---

## Deadfall (Cognitive Honeypots)

### Go Install

```bash
go install github.com/dotsetlabs/deadfall/cmd/deadfall@latest
```

### Download Binary

Download the latest release from [GitHub Releases](https://github.com/dotsetlabs/deadfall/releases).

### Verify Installation

```bash
deadfall --version
```

### Quick Start

```bash
deadfall init
deadfall trap cursor-rules
deadfall serve
```

## Requirements

| Tool | Requirements |
|:-----|:-------------|
| **Hardpoint** | Go 1.22+ (for building from source) |
| **Tollgate** | Node.js 20+ |
| **Deadfall** | Go 1.22+ (for building from source) |

## Updating

```bash
# Hardpoint (reinstall latest)
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest

# Tollgate
npm update -g @dotsetlabs/tollgate

# Deadfall (reinstall latest)
go install github.com/dotsetlabs/deadfall/cmd/deadfall@latest
```

## Uninstalling

```bash
# Hardpoint (remove binary from GOPATH)
rm $(go env GOPATH)/bin/hardpoint

# Tollgate
npm uninstall -g @dotsetlabs/tollgate

# Deadfall (remove binary from GOPATH)
rm $(go env GOPATH)/bin/deadfall
```
