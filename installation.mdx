# Installation

## npm (Recommended)

```bash
npm install -g @dotsetlabs/shield
```

## Yarn

```bash
yarn global add @dotsetlabs/shield
```

## pnpm

```bash
pnpm add -g @dotsetlabs/shield
```

## Verify Installation

```bash
dotset --version
```

## Requirements

- Node.js 20 or later
- npm, yarn, or pnpm

## Project Setup

Shield is designed for zero-migration. If you use `.env` files, you can start using it immediately:

```bash
cd your-project
dotset run -- npm start
```

<Tip>
Shield auto-detects secrets from `.env` files and environment variables. No additional configuration required.
</Tip>

## Updating

```bash
npm update -g @dotsetlabs/shield
```

## Uninstalling

```bash
npm uninstall -g @dotsetlabs/shield
```
