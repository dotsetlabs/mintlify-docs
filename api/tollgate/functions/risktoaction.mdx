# Function: riskToAction()

```ts
function riskToAction(risk, mapping): ConcreteAction;
```

Defined in: src/tollgate/analyzers/index.ts:78

Convert a risk level to a concrete policy action using a risk mapping

## Parameters

### risk

[`RiskLevel`](../type-aliases/RiskLevel.md)

The risk level from analyzer output (e.g., `safe`, `read`, `write`, `destructive`, `dangerous`).

### mapping

[`RiskMapping`](../interfaces/RiskMapping.md) = `DEFAULT_RISK_MAPPING`

Optional mapping configuration that defines how risk levels translate to policy actions.

## Returns

[`ConcreteAction`](../type-aliases/ConcreteAction.md)

The policy action to take based on the risk level (e.g., `allow`, `deny`, `prompt`).
