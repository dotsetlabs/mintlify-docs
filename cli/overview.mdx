# CLI Reference

The shield CLI provides a unified interface for secret protection and cloud analytics.

## Commands

| Command | Description |
|---------|-------------|
| [`run`](/cli/run) | Run commands with protection |
| [`link`](/cli/link) | Connect project to cloud |
| [`secrets`](/cli/secrets) | Manage local encrypted storage |
| [`ci`](/cli/ci) | Run local CI (GitHub Actions) |
| [`auth`](/cli/auth) | Authenticate with cloud |

## Cloud Commands

| Command | Description |
|---------|-------------|
| `dotset link <id> --token <t>` | Connect to dashboard |
| `dotset unlink` | Disconnect from cloud |
| `dotset status` | Show connection status |
| `dotset sync` | Fetch security policies from cloud |

## Global Options

```bash
dotset --version    # Show version
dotset --help       # Show help
```

## Quick Examples

```bash
# Zero-setup protection
dotset run -- npm start

# Redact secrets from output
dotset run --mode redact -- npm test

# Link to cloud for analytics
dotset link proj_123 --token dss_xxx

# Run with telemetry
dotset run --mode redact -- npm start
# Analytics appear at app.dotsetlabs.com

# Disable telemetry for a run
dotset run --no-telemetry -- npm start
```
