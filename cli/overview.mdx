# CLI Reference

The shield CLI provides a unified interface for cross-provider secret management and leak protection.

## Commands

| Command | Description |
|---------|-------------|
| [`secrets init`](/cli/init) | Initialize native encrypted storage |
| [`run`](/cli/run) | Run commands with protection |
| [`secrets`](/cli/secrets) | Manage cross-provider secrets |
| [`ci`](/cli/ci) | Run local CI (GitHub Actions) |
| [`auth`](/cli/auth) | Authenticate with Dotset Cloud |

## Global Options

```bash
dotset --version    # Show version
dotset --help       # Show help
```

## Quick Examples

```bash
# Zero-setup protection
dotset run -- npm start

# Mix and match providers
dotset run --providers dotenv,aws-sm -- node server.js

# Initialize native storage
dotset secrets init

# Set a native secret
dotset secrets set API_KEY "sk-..."

# Run with redaction
dotset run --mode redact -- npm start
```
