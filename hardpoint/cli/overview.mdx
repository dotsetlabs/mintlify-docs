# Hardpoint CLI

Your AI coding assistant can read any file. Have you checked what's in those files?

Hardpoint scans for threats that traditional security tools miss — prompt injections in AI configs, hidden Unicode in code comments, and malicious instructions designed to manipulate AI behavior.

## The Threats Are Real

| Threat | Example | Detection |
| ------ | ------- | --------- |
| **MCPoison** (CVE-2025-54136) | Malicious MCP server configs that hijack AI behavior | AI scanner |
| **Trojan Source** (CVE-2021-42574) | Hidden Unicode that appears invisible but affects AI | AI scanner |
| **Prompt Injection** | "Ignore previous instructions" in README files | AI scanner |
| **Supply Chain** | Malicious instructions in npm dependencies | AI scanner |
| **Shell Backdoors** | rc files modified to exfiltrate data | Shell scanner |
| **Git Hook Attacks** | Pre-commit hooks that run malicious code | Git scanner |

## Installation

```bash
# Using Go
go install github.com/dotsetlabs/hardpoint/cmd/hardpoint@latest

# Using npm (alternative)
npx @dotsetlabs/hardpoint scan
```

## Quick Start

```bash
# Scan everything in 40ms
hardpoint scan

# Focus on AI-specific threats
hardpoint scan ai

# CI/CD mode (JSON output, exit code on findings)
hardpoint scan --output json --severity critical
```

## Commands

| Command | Description |
| ------- | ----------- |
| [`scan`](/hardpoint/cli/scan) | Run security scans on your environment |
| [`integrity`](/hardpoint/cli/integrity) | Verify AI config file integrity |
| [`baseline`](/hardpoint/cli/baseline) | Manage suppression baselines |
| [`fix`](/hardpoint/cli/fix) | Auto-fix certain security findings |

## What We Scan

| Scanner | Targets | Key Detections |
| ------- | ------- | -------------- |
| **AI** | `.cursorrules`, `CLAUDE.md`, `mcp.json`, etc. | Prompt injection, hidden Unicode, role manipulation |
| **Shell** | `.bashrc`, `.zshrc`, shell configs | Environment hijacking, alias poisoning |
| **Git** | `.git/hooks/*`, gitconfig | Malicious hooks, credential theft |
| **Network** | Exposed ports, services | Unauthorized listeners |
| **Secrets** | All files | API keys, tokens, credentials |

## Global Flags

| Flag | Description |
| ---- | ----------- |
| `--config` | Config file path (default: `$HOME/.hardpoint.yaml`) |
| `-o, --output` | Output format: `terminal`, `json`, `sarif` |
| `-v, --verbose` | Enable verbose output |
| `-q, --quiet` | Only output findings |
| `--no-color` | Disable colored output |

## Exit Codes

| Code | Meaning |
| ---- | ------- |
| `0` | No critical findings |
| `1` | Critical findings detected |
