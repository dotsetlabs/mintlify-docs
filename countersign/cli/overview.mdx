# Countersign CLI

Countersign is a supply chain verification tool for MCP servers. It verifies package provenance and publisher identity before installation.

## Installation

```bash
# npm (global)
npm install -g @dotsetlabs/countersign

# npm (local)
npm install @dotsetlabs/countersign
```

## Commands

| Command | Description |
| ------- | ----------- |
| `verify` | Verify package provenance and signatures |
| `sign` | Sign packages for distribution |
| `trust` | Manage trusted publishers |
| `version` | Print version information |

## Global Flags

| Flag | Description |
| ---- | ----------- |
| `-o, --output` | Output format: `terminal`, `json` |
| `-v, --verbose` | Enable verbose output |
| `-q, --quiet` | Only output warnings and errors |
| `--no-color` | Disable colored output |

## Quick Start

```bash
# Verify a package before installation
countersign verify @modelcontextprotocol/server-filesystem

# Show trust score breakdown
countersign verify --detailed @some/mcp-server

# Add a trusted publisher
countersign trust add anthropic

# Sign a package for distribution
countersign sign ./dist
```

## Exit Codes

| Code | Meaning |
| ---- | ------- |
| `0` | Verification passed, package is trustworthy |
| `1` | Verification failed or trust score too low |
| `2` | Package has no provenance information |
