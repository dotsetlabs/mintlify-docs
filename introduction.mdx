## Shield

**Log sanitization for CI/CD.**

Secrets leak in build logs every day. Debug statements, error messages, and verbose logging expose API keys, tokens, and credentials.

**Dotset Shield** intercepts output streams and redacts secrets in real-time — with **zero migration** and **dynamic secret detection**.

```bash
dotset shield run -- npm start
```

> [!NOTE]
> Shield processes secrets locally and never sends values to the cloud. [Learn more →](/shield/security/trust)

---

## Preflight

**Deployment validation.**

Catch configuration issues before they hit production. Preflight runs health probes against your services and validates that everything is ready.

```bash
dotset preflight check
```

---

## Border

**Outbound PII firewall for Node.js.**

Prevent sensitive data from leaking through API requests. Border intercepts `fetch()` and `http.request()` calls and redacts PII before transmission.

```typescript
import { init } from "@dotsetlabs/border";

init({
  destinationPolicies: [
    { domain: "*.openai.com", blockedPiiTypes: ["ssn", "credit_card"] },
  ],
});
```
