## Quick Install

Install all tools with a single package:

```bash
npm install -g @dotsetlabs/cli
```

Or install individual tools:

```bash
npm install -g @dotsetlabs/axion   # Secrets
npm install -g @dotsetlabs/gluon   # Monitoring
npm install -g @dotsetlabs/tachyon # Tunnels
```

## Unified CLI

The `dotset` CLI provides a unified interface for all products:

```bash
# Run with secrets + monitoring
dotset run -- npm start

# Individual product commands
dotset axn set API_KEY "sk-xxx"
dotset gln analyze
dotset tcn share 3000
```
