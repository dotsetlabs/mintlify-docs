# dotset labs

**One CLI for secrets, security, and tunnels.**

dotset is a unified developer platform that combines encrypted secrets management, runtime security telemetry, and zero-trust localhost sharing into a single developer experience.

## Quick Install

```bash
npm install -g @dotsetlabs/cli
```

## The Unified Runtime

With one command, inject secrets and monitor for leaks:

```bash
# Initialize your project
dotset init

# Run with full security stack
dotset run -- npm start
```

## Core Commands

| Command | Description |
|:--------|:------------|
| `dotset init` | Initialize a new project |
| `dotset run` | Run with secrets + monitoring |
| `dotset login` | Authenticate with cloud |
| `dotset status` | Show project status |

## Module Commands

```bash
# Secrets
dotset secrets set API_KEY "sk-..."
dotset secrets list

# Security
dotset scan
dotset sbom --static

# Tunnels
dotset share 3000
dotset tunnels
```
