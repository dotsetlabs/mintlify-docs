# CLI Reference

The Mantle CLI provides a unified interface for secret protection in your CI/CD pipelines.

## Installation

```bash
npm install -g @dotsetlabs/mantle
```

## Commands

| Command | Description |
|---------|-------------|
| [`run`](/mantle/cli/run) | Run commands with secret protection |
| [`scan`](/mantle/cli/scan) | Scan files for secrets |

## Global Options

```bash
mantle --version    # Show version
mantle --help       # Show help
```

## Quick Examples

```bash
# Zero-setup protection
mantle run -- npm start

# Redact secrets from output
mantle run --mode redact -- npm test

# Scan files for secrets
mantle scan src/

# Generate SARIF report for GitHub
mantle scan --format sarif . > results.sarif
```
