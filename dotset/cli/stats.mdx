# dotset stats

Display summary statistics from Tollgate's audit database. This command proxies to `tollgate stats`.

## Usage

```bash
dotset stats [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--audit-path <path>` | Custom path for audit database |

## Examples

```bash
# View statistics
dotset stats

# Use custom audit database
dotset stats --audit-path /custom/path/audit.db
```
