# CLI Reference

The Shield CLI provides a unified interface for secret protection in your CI/CD pipelines.

## Commands

| Command | Description |
|---------|-------------|
| [`run`](/shield/cli/run) | Run commands with protection |

## Global Options

```bash
dotset shield --version    # Show version
dotset shield --help       # Show help
```

## Quick Examples

```bash
# Zero-setup protection
dotset shield run -- npm start

# Redact secrets from output
dotset shield run --mode redact -- npm test

# Generate compliance report
dotset shield run --report -- npm test
```
