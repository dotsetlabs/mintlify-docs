# Axion Quickstart

**The Zero-Disk Secret Plane.** Stop syncing `.env` files. Stream encrypted secrets directly to your app's memory.

## Installation

### CLI

```bash
npm install -g @dotsetlabs/cli
```

### SDK (for programmatic access)

```bash
npm install @dotsetlabs/axion
```

## Initialize a Project

```bash
dotset init --axion
```

This creates:
- `.dotset/axion/key` — Your encryption key (keep this safe!)
- `.dotset/axion/manifest.enc` — Encrypted secrets store

<Warning>
Back up your project key securely. If you lose it, your secrets cannot be recovered.
</Warning>

## Add Secrets

**Set secrets individually:**

```bash
dotset secrets set DATABASE_URL "postgres://localhost/mydb"
dotset secrets set API_KEY "sk-12345" --scope production
```

**Import from .env files:**

```bash
dotset secrets import .env
dotset secrets import .env.production --scope production
```

## Run Your App

```bash
dotset run -- npm start
dotset run --scope production -- node dist/index.js
```

Axion decrypts secrets in memory and injects them into `process.env`.

## Enable Cloud Sync (Optional)

```bash
dotset login           # Authenticate with GitHub/Google
dotset init --cloud    # Create cloud project and link
dotset sync push       # Sync manifest to cloud
```
